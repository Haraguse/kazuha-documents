# 工具栏

![工具栏](/images/settings/toolbar.png)

## 功能详解
| 功能 | 描述 |
| :-: | :-: |
| 顶部工具栏预览区 | 预览工具栏在实际界面中的呈现效果 |
| 显示“清屏” | 调整是否显示“清屏”按钮 |
| 显示“聚光灯” | 调整是否显示“聚光灯”按钮 |
| 显示“计时器” | 调整是否显示“计时器”按钮 |
| 工具栏文字 | 工具项右侧显示功能名称，如“清屏”、“聚光灯”、“计时器”等 |
| 工具栏固定项 | 显示工具栏固定的程序，音乐，图片，快捷方式等。用户可以根据需要添加/删除/重命名固定项。 |


## 关于“如何添加工具栏固定项”
1. 划到“工具栏固定项”区域，点击“添加项”按钮
2. 在弹出的文件选择框中浏览你要固定的内容，支持音乐、图片、软件快捷方式等
3. 点击“打开”按钮，将固定项添加到工具栏固定项区域
4. 你可以根据需要添加/删除/重命名固定项
